カテゴリトップ＞スキンケア＞美容液

よくある質問

Q1.どんな肌タイプに適していますか？ +
A.乾燥肌や混合肌など、さまざまな肌タイプの方にお使いいただけます。使用前にパッチテストをされることをおすすめしています。
Q2.朝晩どちらで使うべきですか？ +
A.朝晩どちらでもご使用いただけます。特に朝は、日焼け止めとの併用を推奨しております。
Q3.敏感肌でも使えますか？ +
A.敏感肌の方でもご利用いただける設計ですが、パッチテストを行ってご確認いただくことをおすすめしています。
Q4.主な成分は？ +
A.ヒアルロン酸やビタミンCが含まれており、保湿や肌のケアをサポートするとされています。
Q5.他のスキンケア製品と併用できますか？ +
A.はい、美容液使用後に保湿クリームなどを組み合わせていただくと、さらにお肌が潤うと感じられる方が多いです。
Q6.保存方法は？ +
A.直射日光や高温多湿を避け、涼しい場所での保管をおすすめしています。
Q7.妊娠中や授乳中の使用は？ +
A.安全のため、事前に医師にご相談いただくと安心です。
Q8.動物実験はしていますか？ +
A.当社では、動物実験を行わない方針で製品を開発しています。
Q9.パッチテストの方法は？ +
A.腕の内側など目立たない部分に少量を塗り、1～2日様子を見て異常がないかご確認ください。
